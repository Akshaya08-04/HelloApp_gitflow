# HelloApp_gitflow

A simple Java Hello World application created to understand and practice
Git and GitFlow branching strategy.

## GitFlow Workflow Used

- main branch – production-ready code
- develop branch – integration branch
- feature branch – new feature development
- release branch – prepare version v1.0.0
- hotfix branch – critical bug fix

## Steps Followed

1. Initialized GitFlow using `git flow init`
2. Created a feature branch and added feature.txt
3. Finished feature and merged into develop
4. Created release v1.0.0 and generated tag
5. Created hotfix branch and applied fix
6. Pushed main, develop, and tags to GitHub
